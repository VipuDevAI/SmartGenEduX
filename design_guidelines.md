# SmartGenEduX Design Guidelines

## Brand Identity
**Website Name:** SmartGenEduX  
**Tagline:** "Smart Automation for Schools" | "Happy Automation. Better Outcomes."  
**Style:** Premium, modern, interactive education platform with Edufy-style aesthetics

## Color Palette
```
Primary Blue: #1F6FE1
Green: #2BB673
Orange Highlight: #F39C12
Light Background: #DFF3F6
Text: #233044
```

## Typography
- **Headings:** Poppins (via Google Fonts)
- **Body:** Inter (via Google Fonts)

## Visual Style System
- **Cards:** Glass-style with soft shadows and rounded corners
- **Gradients:** Smooth transitions (blue → orange for primary actions)
- **Shadows:** Soft, subtle depth
- **Corners:** Consistently rounded throughout
- **Animations:** Smooth scroll, scale on hover, moving elements

## Navigation Bar
Clean, sticky header with:
- Logo (use provided SmartGenEduX logo)
- Links: Home | Products | About | Contact | Login
- Responsive hamburger menu for mobile

## Hero Section
**Layout:** Full-width with large illustration
**Content:**
- Title: "SmartGenEduX — Smart Automation for Schools"
- Subtitle: "Happy Automation. Better Outcomes."
- AI-education themed illustration (large, colorful)
- Two large colorful buttons:
  - "Get Started" (gradient blue → orange)
  - "Explore Products" (gradient green → blue)

**Button Specifications:**
- Pill-shaped, large size
- Gradient fills
- Hover: Scale up (1.05) + enhanced shadow

## Responsive Breakpoints
- **Mobile:** < 768px (single column, stacked)
- **Tablet:** 768px - 1024px (2 columns)
- **Desktop:** > 1024px (full layout, 4 columns for products)

## Interactive Elements
- **Buttons:** Hover scale + shadow enhancement
- **Cards:** Lift on hover
- **Smooth scroll:** Page navigation
- **Animations:** Subtle entrance animations for sections
- **Disabled state:** Muted colors, no hover effects for "Coming Soon"

- **Feature cards:** Simple icons matching brand colors
